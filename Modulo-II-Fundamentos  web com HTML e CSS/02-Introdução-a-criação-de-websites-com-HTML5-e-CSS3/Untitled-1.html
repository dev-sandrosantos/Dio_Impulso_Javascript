O QUE É HTML?
O HTML é uma linguagem de marcação de hipertexto usada para criar sites de forma geral e páginas da web. O código usado para torná-los visualmente atraentes é conhecido como CSS mas não vamos falar sobre esse. Por enquanto, vamos nos concentrar em ensinar a você como construir, em vez de projetar. Como dito anteriormente, o HTML é uma Linguagem de Marcação de HyperTexto e, para entender o HTML, é preciso entender antes o que é HyperTexto e uma Linguagem de Marcação.
